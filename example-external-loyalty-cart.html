Implementation Details

To integrate the LoyaltyEngage cart functionality into your external HTML:

  1. Make sure your HTML is loaded within a Shopware page where the LoyaltyEngage plugin is active
  2. Create buttons or links with onclick handlers that call the redeemWithPoints function
  3. Pass the product ID to the redeemWithPoints function
  4. Handle the success or error callback as needed

JavaScript Code Example:

function redeemWithPoints(productId) {
    if (typeof window.addProductToLoyaltyCart === 'function') {
        window.addProductToLoyaltyCart(productId, function(success, message) {
            if (success) {
                // Handle success
                console.log('Product added successfully:', message);
            } else {
                // Handle error
                console.error('Failed to add product:', message);
            }
        });
    } else {
        console.error('LoyaltyEngage plugin is not loaded');
    }
}
